Sistema de Gestión de Citas Médicas
Integrante:
Fernando Ruiz Alvarado 
El proyecto consiste en el desarrollo de un sistema web que permite administrar el proceso de agendamiento de citas médicas en un centro de salud o clínica. El sistema facilita la interacción entre pacientes, médicos y personal administrativo, permitiendo registrar información de usuarios, gestionar horarios de atención, agendar y cancelar citas, así como llevar un registro digital del historial clínico de cada consulta.
